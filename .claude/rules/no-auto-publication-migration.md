# Sem publicação automática

- Registos migrados permanecem `preliminary`.
- Media permanece `publicationAllowed: false`.
- Direitos permanecem `pending-review` até prova registada.
- Traduções de rascunho não podem ser publicadas.
- `MM202617` permanece bloqueado até decisão explícita.
