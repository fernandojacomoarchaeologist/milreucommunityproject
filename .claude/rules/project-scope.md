# Escopo do projeto

- Tratar o Projeto Comunitário de Milreu como programa guarda-chuva.
- Não tratar o Museu de Memórias como totalidade do projeto.
- Associar cada funcionalidade à iniciativa e ao produto correspondentes.
- Executar `scope-check` antes de propor implementação.
- Não criar nova visualização sem registo e spec.
- Não desenvolver iniciativas futuras além do que estiver aprovado para o pacote atual.
