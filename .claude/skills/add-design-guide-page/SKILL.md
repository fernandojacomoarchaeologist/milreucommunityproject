# Skill — Adicionar página ao guia

1. Confirmar grupo e problema documental.
2. Verificar se existe spec ou decisão de suporte.
3. Adicionar registo ao catálogo canónico.
4. Gerar a projeção JavaScript.
5. Definir estado inicial adequado.
6. Acrescentar exemplo apenas quando sustentado.
7. Atualizar pesquisa, testes e release.
8. Verificar teclado, responsive e quatro idiomas da interface.
