# Skill — analisar fonte visual

## Finalidade

Analisar uma página ou conjunto de páginas da fonte sem inventar decisões finais.

## Procedimento

1. Identificar página PDF e página impressa, quando visível.
2. Descrever composição, cor, tipografia funcional, imagens, plantas, objetos e legendas.
3. Separar observação de interpretação.
4. Classificar cada elemento como `preserve`, `adapt`, `reference-only`, `do-not-reproduce` ou `pending-rights`.
5. Registar problemas de acessibilidade.
6. Propor componentes candidatos, sem os declarar aprovados.
7. Atualizar `page-elements.json` e o MD correspondente.
8. Não extrair token final, logótipo ou fonte por aproximação.
9. Não publicar a página.
10. Registar release da alteração.
