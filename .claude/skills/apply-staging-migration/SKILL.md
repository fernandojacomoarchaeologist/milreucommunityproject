# Skill — Aplicar em staging

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Confirmar ambiente staging.
2. Confirmar credenciais fora do log.
3. Aplicar migrations versionadas.
4. Executar smoke tests.
5. Registar resultado.
6. Não promover automaticamente a produção.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
