# Skill — Auditar o guia

Verificar:

- rotas e links;
- títulos e landmarks;
- foco e teclado;
- pesquisa;
- drawer mobile;
- idiomas;
- estados de maturidade;
- uso de tokens;
- ausência de dados reais inventados;
- ausência de fontes ou imagens não autorizadas;
- copyright e release.
