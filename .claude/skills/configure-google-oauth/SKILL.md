# Skill — Configure Google Oauth

## Procedimento

1. Criar ou selecionar o projeto Google Cloud.
2. Configurar audiência e consentimento.
3. Criar cliente OAuth Web.
4. Ativar Google no Supabase.
5. Adicionar callbacks local, Pages e domínio.
6. Gerar configuração runtime sem service role.
7. Testar login e logout.
