# Skill — Alterar token de design

## Quando usar
Ao criar, renomear ou alterar um token.

## Procedimento
1. identificar o problema e o papel semântico;
2. localizar a decisão e o token atual;
3. avaliar impacto em Portal, Museu, impressão e Flutter;
4. alterar a fonte consolidada;
5. atualizar ficheiros por domínio e exportações;
6. executar contraste e paridade;
7. atualizar release e, se estrutural, criar ADR;
8. não modificar componentes não relacionados.
