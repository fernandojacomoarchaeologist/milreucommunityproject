# Skill — Exportar snapshot público

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Usar query aprovada.
2. Excluir campos internos.
3. Normalizar ordem e idiomas.
4. Validar schema.
5. Gerar manifest e checksum.
6. Não editar o resultado manualmente.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
