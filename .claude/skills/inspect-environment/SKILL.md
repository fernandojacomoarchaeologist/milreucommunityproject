# Skill — Inspecionar ambiente

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Identificar `MILREU_ENV`.
2. Verificar Git, Node, Docker e Supabase CLI.
3. Confirmar branch e estado do working tree.
4. Mascarar qualquer valor sensível.
5. Produzir relatório sem alterar o ambiente.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
