# Skill — Gerir revisão de memória

## Quando usar

Quando uma memória recebe correção, identificação, nova fonte, tradução, pedido de direitos ou alteração editorial.

## Procedimento

1. Preservar a versão anterior.
2. Criar nova revisão.
3. Registar motivo e campos alterados.
4. Associar evidências e pedido relacionado.
5. Reavaliar certeza, direitos e publicação.
6. Solicitar revisão humana quando a alteração afetar identidade, direitos, datação ou interpretação.
7. Publicar apenas após aprovação explícita.
