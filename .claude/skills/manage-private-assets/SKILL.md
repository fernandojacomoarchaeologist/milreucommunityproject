# Skill — Gerir fontes privadas

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Confirmar storage autorizado.
2. Recuperar sem expor URL.
3. Guardar apenas em caminho ignorado.
4. Verificar checksum.
5. Não copiar para pasta pública.
6. Atualizar estado local.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
