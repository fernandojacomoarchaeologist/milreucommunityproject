# Skill — Exportar tokens por plataforma

## Procedimento
1. ler `tokens.json`;
2. manter nomes semânticos equivalentes;
3. gerar CSS, JavaScript e Flutter;
4. não incluir binários de fontes;
5. manter avisos de impressão;
6. executar `platform-parity.test.mjs`;
7. registar diferenças inevitáveis de plataforma.
