# Skill — Preparar Gate A

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Gerar índice de páginas críticas.
2. Preparar capturas desktop, tablet e mobile.
3. Não decidir pelo utilizador.
4. Atualizar `SCREENSHOT_INDEX.md`.
5. Manter estado máximo `validated-for-architecture`.
6. Listar questões abertas.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
