# Skill — Preparar deployment produtivo

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Confirmar pedido explícito do utilizador.
2. Exigir change ticket.
3. Exigir confirmação literal.
4. Executar preflight.
5. Acionar workflow protegido.
6. Aguardar required reviewer.
7. Não contornar o ambiente.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
