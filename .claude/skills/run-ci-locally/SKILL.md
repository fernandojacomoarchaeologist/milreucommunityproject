# Skill — Executar CI local

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Instalar dependências.
2. Executar validação de infraestrutura.
3. Executar testes JS.
4. Executar validação de schemas.
5. Executar testes de banco quando Docker estiver disponível.
6. Resumir falhas.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
