# Skill — Preparar ambiente local

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Inspecionar versões e ficheiros existentes.
2. Instalar dependências usando o lockfile.
3. Criar `.env.local` a partir do exemplo sem preencher segredos.
4. Validar Docker e Supabase CLI.
5. Executar `infra:validate` e `infra:test`.
6. Não ligar a projeto remoto.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
