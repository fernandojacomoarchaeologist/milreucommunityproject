# Skill — Testar banco local

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Confirmar que o ambiente é local.
2. Iniciar Supabase local.
3. Executar reset completo.
4. Executar testes SQL.
5. Validar RLS.
6. Encerrar serviços quando apropriado.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
