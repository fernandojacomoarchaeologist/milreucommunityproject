# Skill — Traduzir registo de memória

## Quando usar

Para produzir ou rever conteúdo em `en`, `es` ou `fr`.

## Procedimento

1. Usar `pt-PT` como fonte.
2. Preservar nomes, datas, códigos e graus de certeza.
3. Usar o glossário controlado quando existir.
4. Marcar saída automática como `machine-draft`.
5. Não marcar como `reviewed` ou `approved` sem revisão humana.
6. Não preencher por interpretação informação ausente no original.
7. Registar versão e revisor quando aplicável.
