# Skill — Validar snapshot

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Validar JSON.
2. Validar schema.
3. Verificar ausência de campos proibidos.
4. Verificar versionamento e checksum.
5. Comparar contagens com a origem.
6. Bloquear publicação em caso de divergência.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
