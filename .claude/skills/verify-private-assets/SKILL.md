# Skill — Verificar fontes privadas

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Ler manifests.
2. Verificar existência opcional.
3. Comparar hashes.
4. Validar miniaturas esperadas.
5. No modo público, não falhar pela ausência.
6. Relatar divergências.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
