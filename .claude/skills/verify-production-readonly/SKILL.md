# Skill — Consultar produção com segurança

## Objetivo

Executar este procedimento de forma segura e auditável.

## Procedimento

1. Confirmar finalidade e mínimo necessário.
2. Usar views aprovadas.
3. Não selecionar dados pessoais desnecessários.
4. Não executar escrita.
5. Não persistir resultados sensíveis em ficheiros.
6. Registar consulta de auditoria quando aplicável.

## Saída obrigatória

- ações executadas;
- ações não executadas;
- ambiente;
- validações;
- riscos;
- próxima decisão humana.
