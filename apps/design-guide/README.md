# Guia Visual — versão 0.4.0

Aplicação estática do Sistema de Design de Milreu com fundações, identidade, componentes e padrões.

## Executar

A partir da raiz do repositório:

```bash
python3 -m http.server 8080
```

Abrir `http://localhost:8080/apps/design-guide/`.

## Dependências

- `packages/design-tokens/v0.2/`;
- `packages/design-components/v0.4/`;
- `packages/design-patterns/v0.4/`;
- `packages/design-guide-catalog/v0.4/`.

Todos os exemplos são demonstrativos e não pertencem ao acervo.

## Ferramentas relacionadas

`apps/foundations-lab/` (Pacote 05B) é ferramenta interna e não deve ser ligada ao menu público.

## Atualização (Pacote 05D)

Substituição controlada da versão 0.3.0 pela 0.4.0: o shell, pesquisa, rotas e navegação do 05C são preservados; foram adicionados registos, renderers, estilos e páginas de componentes e padrões.

## Atualização (Pacote 05E)

Guia atualizado para 0.5.0: adicionadas seis rotas de identidade e conhecimento (marca, logótipo, dark mode, iconografia, aplicações e relação com Milreu Proteus), preservando as 58 rotas existentes. Ativos de marca permanecem `draft`.
