# Exemplos

`MM202601.preliminary.example.json` é um mapeamento demonstrativo de um registo real da versão preliminar.

## Regras de utilização

- não constitui migração oficial;
- não confirma publicação;
- não resolve direitos;
- não aprova a tradução inglesa;
- não deve ser copiado como conteúdo factual para outros registos;
- serve apenas para testar estrutura e demonstrar a correspondência com o legado.
