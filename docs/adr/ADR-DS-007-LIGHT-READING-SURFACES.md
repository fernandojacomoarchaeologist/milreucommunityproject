# ADR-DS-007 — Superfícies claras de leitura

## Estado
Aceite.

## Decisão
Textos longos usam branco documental ou papel claro. Preto e vermelho são superfícies de curta duração.

## Consequências
- acessibilidade e leitura prolongada prevalecem;
- texturas ficam subordinadas;
- o Museu pode usar full-bleed, mas detalhes textuais devem ter base legível.
