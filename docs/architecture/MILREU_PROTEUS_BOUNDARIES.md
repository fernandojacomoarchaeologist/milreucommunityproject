# Limites do Milreu Proteus

Milreu Proteus é a camada estruturada de conhecimento e dados do Projeto Comunitário de Milreu.

## Inclui

- entidades e relações;
- estados editoriais;
- proveniência;
- fontes;
- direitos;
- revisões;
- dados de participação;
- datasets públicos;
- exportações;
- histórico.

## Não inclui automaticamente

- originais protegidos;
- consentimentos expostos ao público;
- credenciais;
- dados pessoais sem necessidade;
- interpretações não revistas;
- conteúdo demonstrativo tratado como real.

## Regra

As interfaces consomem Proteus; não duplicam manualmente a mesma informação em várias bases.
