# Arquitetura de conteúdo do Portal

O Portal orienta todo o Projeto Comunitário de Milreu.

## Camadas

- apresentação do projeto;
- metodologia;
- iniciativas;
- Museu;
- Milreu Proteus;
- participação;
- contexto institucional.

## Fonte

O ficheiro `portal-content.json` é uma etapa estática. No futuro, os mesmos campos serão fornecidos pelo Milreu Proteus.

## Totens

As iniciativas e memórias devem manter identificadores e descrições canónicas para permitir variantes curtas destinadas aos totens.
