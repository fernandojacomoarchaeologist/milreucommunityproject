# Arquitetura de marca

A marca principal é **Projeto Comunitário de Milreu**, programa guarda-chuva. Museu de Memórias, Open Milreu e outras frentes são iniciativas subordinadas. Milreu Proteus é um produto técnico de conhecimento e dados. Nenhuma destas camadas substitui o nome principal.
