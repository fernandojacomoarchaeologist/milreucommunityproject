# Favicon e ícones de aplicação

O símbolo isolado é a base para favicon e ícones. Ícones de aplicação usam fundo chapado claro ou escuro; não usam gradientes. A futura aplicação Flutter deve reutilizar a mesma fonte de ativos e não redesenhar o símbolo.
