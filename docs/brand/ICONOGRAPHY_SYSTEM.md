# Sistema de iconografia

Os ícones usam grelha 24×24, traço 1,75, terminais arredondados e `currentColor`. A iconografia funcional deve ser neutra. A iconografia de domínio pode dialogar com tesselas, círculos e plantas, sem recorrer a estereótipos romanos sem função.
