# Sistema do logótipo

O logótipo é composto por símbolo, assinatura `PROJETO COMUNITÁRIO` e denominação `DE MILREU`. A composição vertical é a versão principal. Não alterar proporções, alinhamento, hierarquia ou desenho. Os derivados transparentes são `draft`.
