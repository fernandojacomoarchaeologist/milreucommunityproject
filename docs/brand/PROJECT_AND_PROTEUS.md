# Projeto e Milreu Proteus

Proteus organiza registos canónicos, fontes, proveniência, cronologia, glossário, dados e relações. O Portal, o Museu, os buscadores, os sistemas de IA e a futura aplicação consomem essa camada. Proteus não recebe logótipo próprio nesta fase.
