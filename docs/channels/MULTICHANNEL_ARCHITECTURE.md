# Arquitetura multicanal

Uma memória é uma entidade canónica.

```text
Memória
├── dados factuais
├── fotografia
├── narrativa
├── contexto
├── fontes
├── crédito
├── direitos
└── apresentações
    ├── Portal
    ├── Museu
    ├── Totem
    └── Painel
```

Os canais não duplicam manualmente os factos.
