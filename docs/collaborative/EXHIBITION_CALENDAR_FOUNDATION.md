# Fundação do calendário de exposições

O 08A prepara quatro entidades:

```text
Exposição
→ Local
→ Agendamento
→ Estado
```

## Tipos de exposição

- fixa;
- itinerante;
- temporária;
- digital.

## Local

Pode representar:

- museu;
- escola;
- biblioteca;
- centro cultural;
- universidade;
- espaço público;
- outro.

## Agendamento

Regista:

- início e fim;
- instalação;
- desmontagem;
- estado;
- notas públicas;
- notas internas.

## Limite do 08A

O banco e o empty state estão preparados. O calendário visual, o CRUD completo, conflitos de datas, notificações e publicação serão desenvolvidos noutro pacote.
