# Perfil, função e permissão

São dimensões diferentes.

## Perfil principal

Descreve o vínculo predominante:

- Coordenação;
- Voluntário;
- Investigador;
- Colaborador comunitário;
- Parceiro institucional;
- Revisor;
- Tradutor;
- Observador.

O cadastro contém obrigatoriamente:

- nome;
- e-mail da conta Google;
- perfil principal.

## Funções

Uma pessoa pode acumular funções, por exemplo:

```text
Perfil principal: Investigador
Funções: researcher + reviewer + translator
```

`master` é função de sistema, não perfil público.

## Permissões

As funções concedem permissões específicas. A interface oculta módulos não autorizados, mas a proteção real permanece no banco por RLS.
