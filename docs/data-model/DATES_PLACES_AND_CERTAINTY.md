# Datas, lugares e certeza

## Datas

Uma data possui:

- texto de apresentação localizado;
- início e fim normalizados, quando conhecidos;
- precisão: dia, mês, ano, década, século ou desconhecida;
- certeza;
- fontes relacionadas.

A data de revelação de uma fotografia não deve ser confundida com a data do evento representado.

## Lugares

Cada lugar pode ter:

- designação pública;
- designação específica;
- identificador normalizado;
- precisão espacial;
- coordenadas opcionais;
- classificação de sensibilidade;
- decisão explícita sobre publicação.

Coordenadas de residências, propriedades privadas ou locais sensíveis devem ser omitidas, aproximadas ou marcadas como `withheld`.

## Certeza

Níveis autorizados:

- `certain`;
- `probable`;
- `hypothesis`;
- `unknown`;
- `not-assessed`.

Não existe o nível `mixed`. Quando diferentes partes de um registo têm certezas distintas, cada afirmação ou entidade deve possuir o seu próprio nível.
