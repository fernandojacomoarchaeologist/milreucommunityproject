# Identificadores e referências

## Memórias

Formato atual: `MM` seguido de seis algarismos, por exemplo `MM202601`.

O identificador é persistente e não deve mudar com título, datação, idioma ou publicação.

## Entidades auxiliares

Prefixos recomendados:

- `ASSET-` — ativo de media;
- `SRC-` — fonte;
- `PROV-` — evento de proveniência;
- `CONTRIB-` — contribuição comunitária;
- `CONCERN-` — pedido de direitos, correção ou remoção;
- `PLACE-` — lugar normalizado;
- `PERSON-` — pessoa ou grupo, quando existir registo próprio;
- `REV-` — revisão.

## Referências externas

Identificadores externos devem conservar o sistema de origem, por exemplo códigos arquivísticos, DOI, ISBN, inventários institucionais ou URLs persistentes.

## Dados privados

Referências a contactos, consentimentos ou documentos privados devem apontar para um identificador seguro externo; não devem copiar o conteúdo sensível para o registo público.
