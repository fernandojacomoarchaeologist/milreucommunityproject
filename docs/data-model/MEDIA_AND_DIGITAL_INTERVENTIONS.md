# Media e intervenções digitais

## Tipos de ativo

- original privado;
- master de preservação;
- derivado público;
- miniatura;
- versão para impressão;
- transcrição;
- áudio ou vídeo relacionado.

## Separação de armazenamento

O Git pode guardar derivados públicos e manifests. Originais de alta resolução, consentimentos e ficheiros sensíveis devem permanecer em armazenamento apropriado.

## Intervenções digitais

Qualquer restauração, correção, recorte, colorização, reconstrução ou uso de IA deve ser declarado com:

- tipo de intervenção;
- descrição;
- ferramenta, se conhecida;
- data;
- referência ao ativo de origem;
- referência a prompt ou procedimento, quando aplicável;
- estado de revisão humana;
- indicação de alteração substantiva ou não substantiva.

Uma intervenção digital não pode ser apresentada como original histórico.
