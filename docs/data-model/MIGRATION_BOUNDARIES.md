# Limites da migração

## Pacote 03

Define estrutura, validação, vocabulários, templates e exemplo de mapeamento.

## Pacote 04

Deverá:

- inventariar os 31 registos;
- converter um registo de cada vez;
- preservar textos originais;
- produzir relatório de lacunas e ambiguidades;
- não publicar automaticamente;
- manter reversibilidade e ligação ao legado;
- validar todos os registos;
- separar dados demonstrativos de dados reais;
- identificar inconsistências sem as corrigir por invenção.

## Fora do âmbito

- base de dados;
- moderação online;
- interface pública;
- tradução em massa;
- resolução jurídica de direitos;
- revisão científica final dos conteúdos.
