# Proveniência e fontes

## Fontes

Fontes podem incluir:

- arquivo;
- publicação;
- fotografia física;
- testemunho oral;
- reunião comunitária;
- formulário;
- página institucional;
- rede social;
- investigação de campo;
- catálogo museológico;
- documento do projeto.

Cada fonte deve indicar o que sustenta e, quando necessário, páginas, códigos, datas de consulta ou localizadores.

## Proveniência

A proveniência documenta eventos sobre o próprio registo ou ativo:

- fornecimento;
- aquisição;
- digitalização;
- catalogação;
- identificação;
- validação;
- restauração;
- tradução;
- revisão de direitos;
- publicação;
- retirada.

Fonte e proveniência não são sinónimos: uma fonte sustenta uma afirmação; a proveniência descreve a trajetória documental do registo ou ativo.
