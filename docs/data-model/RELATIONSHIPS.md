# Relações entre registos

Relações ligam memórias a:

- outras memórias;
- pessoas ou grupos;
- lugares;
- eventos;
- estudos;
- artefactos;
- iniciativas;
- exposições.

Cada relação deve indicar tipo, alvo, certeza, direção e notas. Relações não devem ser inferidas apenas por tags coincidentes.

Tipos iniciais incluem:

- `related-to`;
- `same-event-as`;
- `same-place-as`;
- `depicts`;
- `mentions`;
- `documents`;
- `derived-from`;
- `part-of-series`;
- `contrasts-with`;
- `supports`.

O vocabulário é extensível por decisão documentada.
