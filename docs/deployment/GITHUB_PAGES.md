# GitHub Pages

## Build

```bash
npm run build
```

Publicar o conteúdo de `dist/`.

A aplicação usa hash routing, por isso funciona em subdiretórios de GitHub Pages sem configuração de servidor.

Durante o 07A, `robots.txt` bloqueia indexação.
