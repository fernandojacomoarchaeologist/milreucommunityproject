# Uso de imagens e atribuição

## Princípio

A presença de uma imagem no PDF não significa que o projeto possua direitos de publicação dessa imagem.

## Estados

- `internal-reference`: consulta interna apenas;
- `rights-review`: requer verificação;
- `licensed-for-use`: autorização documentada;
- `public-domain`: verificação documentada;
- `do-not-publish`: bloqueada.

## Regra para o quadro visual

O quadro visual deste pacote é interno. Quando o catálogo público do Design System for criado, deverá utilizar exemplos próprios, abstrações, diagramas originais ou imagens explicitamente autorizadas.

## Atribuição mínima

Quando a fonte for mencionada:

HAUSCHILD, Theodor; TEICHNER, Felix — *Milreu: Ruínas*. Lisboa: Instituto Português do Património Arquitectónico, 2002.
