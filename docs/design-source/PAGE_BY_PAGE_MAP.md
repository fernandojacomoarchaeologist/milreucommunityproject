# Mapa visual página a página

Este mapa usa a numeração do PDF digitalizado. A versão estruturada está em `data/design-source/page-elements.json`.

## PDF 01 — Capa

- Elementos: opening, cover, red, plan, photography.
- Observação: Red signature field; overlaid archaeological plan; framed ruins photograph; institutional marks.
- Decisão: `adapt`.
- Nota: High-impact opening; avoid using red as default reading background.
