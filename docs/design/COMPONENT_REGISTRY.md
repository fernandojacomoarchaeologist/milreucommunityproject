## Critérios para `stable`

- contrato documentado;
- exemplos no guia;
- teclado e foco verificados;
- contraste verificado;
- quatro idiomas considerados;
- comportamento responsivo verificado;
- utilização real em pelo menos um produto;
- alteração registada em release.

## Componentes não autorizados nesta versão

- carrossel automático;
- timeline final;
- mapa de narrativas final;
- visualizador PDF;
- tabelas de dados finais;
- editor de narrativas;
- sistema de autenticação;
- upload de fotografias;
- componentes Flutter de produção.

Esses itens requerem specs e pacotes próprios.
