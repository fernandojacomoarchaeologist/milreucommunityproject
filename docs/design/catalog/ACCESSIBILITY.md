# Acessibilidade do catálogo

Baseline:

- skip link;
- landmarks semânticos;
- navegação por teclado;
- foco visível;
- diálogo nativo de pesquisa;
- títulos hierárquicos;
- contraste baseado nos tokens 05B;
- `prefers-reduced-motion`;
- targets mínimos adequados;
- idioma do documento atualizado pelo seletor.

A interface traduzida não implica que todo o conteúdo documental esteja traduzido.
