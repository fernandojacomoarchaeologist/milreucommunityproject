# Política de exemplos de código

Exemplos devem:

- usar tokens, não valores arbitrários;
- ser mínimos e executáveis;
- indicar dependências;
- não incluir dados pessoais ou conteúdos reais não publicados;
- não copiar código de design systems externos;
- manter copyright do projeto.

Snippets no catálogo não substituem os ficheiros fonte.
