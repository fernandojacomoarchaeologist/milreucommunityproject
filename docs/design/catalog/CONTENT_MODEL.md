# Modelo de conteúdo

O catálogo é declarativo. Cada página possui `id`, `group`, `slug`, `title`, `summary`, `status`, `tags`, `preview`, `source`, `platform` e `sections`.

O conteúdo visível no guia não substitui os MDs de referência. O JSON serve à navegação e à visualização; decisões extensas continuam documentadas em Markdown e ADRs.
