# Modelo de maturidade

Estados permitidos:

- `proposed` — ideia registada;
- `draft` — solução exploratória;
- `in-review` — decisão em revisão;
- `validated` — base validada para construção interna;
- `approved` — autorizada para produção;
- `deprecated` — em retirada.

A transição para `approved` exige revisão explícita e release. Visibilidade no catálogo não equivale a aprovação.
