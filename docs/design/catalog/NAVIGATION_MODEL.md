# Modelo de navegação

A navegação principal é lateral em ecrãs amplos e torna-se drawer em ecrãs reduzidos. Os grupos são estáveis: Introdução, Fundações, Identidade, Componentes, Padrões, Experiências e Governação.

Cada página possui URL por hash, breadcrumb, índice local e ligações anterior/seguinte. A pesquisa oferece acesso transversal sem substituir a hierarquia.
