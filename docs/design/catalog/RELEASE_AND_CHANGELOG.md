# Releases e changelog

O catálogo segue o ciclo 0.x até a validação do sistema completo. Toda alteração deve indicar:

- páginas adicionadas ou removidas;
- estados alterados;
- rotas afetadas;
- tokens consumidos;
- migração necessária;
- testes executados;
- decisões em aberto.
