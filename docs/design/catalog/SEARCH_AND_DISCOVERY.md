# Pesquisa e descoberta

A pesquisa local indexa título, resumo, grupo e tags. O atalho `/` abre o diálogo, setas percorrem resultados e `Enter` abre a página.

Não existe telemetria nesta versão. Termos pesquisados não são enviados para serviços externos.
