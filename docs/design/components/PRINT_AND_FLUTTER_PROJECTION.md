# Projeções para impressão e Flutter

As projeções reutilizam semântica e tokens, mas não são cópias literais do CSS. Impressão exige escala física, sangria, materiais, QR e preflight. Flutter exige API comum, widgets acessíveis e paridade de estados. Nenhuma das projeções está implementada neste pacote.
