# Componentes de direitos e contestação

O aviso de direitos e o pedido de correção ou remoção devem estar próximos do conteúdo afetado. O sistema deve comunicar que o processo é comunitário e vivo, sem prometer remoção automática nem atribuir todos os direitos ao projeto.
