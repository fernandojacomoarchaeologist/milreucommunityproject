# Linha de base de acessibilidade

- contraste conforme WCAG AA no mínimo;
- zoom até 200% sem perda de conteúdo;
- foco visível;
- controlos com alvo mínimo de 44 × 44 px quando aplicável;
- navegação por teclado;
- redução de movimento;
- alt text e legendas independentes;
- idioma da página e dos trechos identificados;
- não depender de cor, posição ou gesto único;
- ordem de leitura linear em mobile;
- tipografia e medida ajustadas para leitura prolongada.
