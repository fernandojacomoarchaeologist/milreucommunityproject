# Matriz de utilização de cor

| Contexto | Fundo | Texto | Observações |
|---|---|---|---|
| leitura longa | `surface.reading` | `text.primary` | padrão principal |
| enquadramento | `surface.canvas` | `text.primary` | papel ligeiramente quente |
| abertura | `surface.signature` | `text.onSignature` | curta duração |
| destaque escuro | `surface.inverse` | `text.inverse` | blocos breves |
| memória provável | claro | `certainty.probable` + rótulo | cor não é único sinal |
| fonte institucional | claro | texto + filete vermelho | evitar grandes massas |
| mapa | `map.background` | paleta funcional | sem associação automática à marca |

## Proibições

- vermelho atrás de parágrafos extensos;
- texto vermelho claro sobre papel claro;
- preto absoluto `#000000` em novos componentes;
- aplicação de sépia para simular idade;
- cores de dados sem legenda textual.
