# Cores de dados e mapas

A paleta funcional não representa a identidade do projeto.

| Papel | Cor |
|---|---:|
| água/informação | `#315D6D` |
| vegetação/positivo | `#2F6B4F` |
| via/atenção | `#8B6A1F` |
| categoria histórica | `#74546A` |
| estrutura principal | `#7A5B3A` |
| área neutra | `#8D8170` |

## Regras

- oferecer padrões, rótulos ou símbolos além da cor;
- não usar a paleta para indicar certeza sem mapeamento explícito;
- garantir contraste em linhas, pontos e legendas;
- mapas públicos não devem revelar coordenadas sensíveis;
- categorias devem manter a mesma associação dentro da mesma visualização.
