# Sistema de fotografia

## Modos

1. **Documental:** imagem integral, sem corte significativo.
2. **Editorial:** corte autorizado para composição, mantendo registo do original.
3. **Imersivo:** imagem em grande escala, com opção de ver integralmente.
4. **Objeto:** recorte sobre fundo neutro, com escala e proveniência quando disponíveis.

## Regras

- não aplicar sépia ou granulação para simular antiguidade;
- documentar restauro, recorte, colorização ou geração assistida;
- manter original e derivados separados;
- não eliminar pessoas ou elementos históricos sem nota;
- alt text deve descrever o que é visível e evitar interpretações não confirmadas;
- overlays de texto devem ter área de contraste controlada, nunca depender da fotografia.
