# Fundações para impressão

## Estado

Os valores de ecrã são aprovados; equivalentes CMYK e materiais exigem prova física. Não inferir Pantone a partir do scan.

## Regras

- vermelho de assinatura deve ser provado no suporte final;
- corpo para painéis deve ser calibrado pela dimensão física e distância de leitura;
- preto rico apenas quando tecnicamente apropriado; texto pequeno deve usar preto seguro definido pela gráfica;
- imagens devem conservar perfil e resolução adequados;
- QR deve respeitar área livre, contraste e tamanho de teste;
- texturas não podem produzir banding ou reduzir legibilidade;
- ficheiro final exige preflight específico no pacote de exposição física.
