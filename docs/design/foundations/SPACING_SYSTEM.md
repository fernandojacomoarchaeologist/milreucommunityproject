# Sistema de espaçamento

A escala parte de 4 px e utiliza passos nomeados. Componentes devem consumir tokens.

| Token | Valor |
|---|---:|
| `space.1` | 4 px |
| `space.2` | 8 px |
| `space.3` | 12 px |
| `space.4` | 16 px |
| `space.5` | 20 px |
| `space.6` | 24 px |
| `space.8` | 32 px |
| `space.10` | 40 px |
| `space.12` | 48 px |
| `space.16` | 64 px |
| `space.20` | 80 px |
| `space.24` | 96 px |
| `space.32` | 128 px |

Espaços de composição editorial usam múltiplos maiores; microalinhamentos usam 4–12 px.
