# Sistema de texturas

Texturas são opcionais e subordinadas à leitura.

- textura de papel: ruído muito discreto, até 3% de opacidade;
- linhas de planta: podem funcionar como marca de abertura, com baixa opacidade;
- padrões de mosaico: não usar como wallpaper repetitivo;
- manchas, vincos e sombras do scan: não reproduzir;
- textura nunca atrás de corpo longo quando reduzir contraste;
- CSS deve oferecer alternativa sem textura.
