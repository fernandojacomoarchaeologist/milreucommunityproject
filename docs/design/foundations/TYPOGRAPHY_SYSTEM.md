# Sistema tipográfico de produção

## Famílias por função

| Função | Família | Uso |
|---|---|---|
| display | Fraunces | aberturas, títulos expressivos e números editoriais |
| editorial | Spectral | narrativas, artigos, contexto e textos longos |
| interface | Archivo | navegação, filtros, controlos, tabelas e metadados |
| código | monoespaçada do sistema | exemplos técnicos no guia |

## Regras

- Fraunces não deve ser usada em corpo longo;
- Spectral deve partir de 18 px na leitura principal;
- Archivo deve ser usado em controlos e informação funcional;
- itálico verdadeiro para termos latinos e títulos de obras;
- não simular condensação, negrito ou itálico;
- máximo recomendado de 68 caracteres por linha em leitura contínua;
- os quatro idiomas devem ser testados antes de publicação.
