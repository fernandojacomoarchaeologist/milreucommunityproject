# Composição orientada pela comunidade

No detalhe de memória, a sequência preferencial é fotografia, narrativa comunitária, descrição objetiva, contexto, proveniência, fontes, relações e direitos. O contexto institucional não deve anteceder ou reescrever a voz comunitária.
