# Fluxo de desenvolvimento

```text
issue ou decisão
→ branch
→ migration/código
→ testes locais
→ CI
→ revisão
→ staging
→ release
→ produção protegida
```

Alterações de dados públicos acrescentam:

```text
export
→ schema
→ revisão editorial
→ snapshot
→ publicação
```
