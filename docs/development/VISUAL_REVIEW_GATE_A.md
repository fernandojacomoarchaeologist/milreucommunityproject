# Gate A de revisão visual

## Escopo

- assinatura visual;
- vermelho e preto;
- fundos;
- tipografia;
- logótipo;
- menu e drawer;
- cartão de memória;
- visualizador;
- metadados;
- direitos;
- idioma;
- filtros;
- estados vazios.

## Resultado

Cada item recebe:

- accepted-for-architecture;
- revise;
- blocked;
- not-reviewed.

O sistema não deve transformar este gate em aprovação final.
