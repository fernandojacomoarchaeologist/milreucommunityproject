# Classificação dos conteúdos

Cada registo migrado mantém:

- conteúdo factual legado;
- conteúdo interpretativo legado;
- texto comunitário ou atribuído à comunidade;
- contexto histórico;
- crédito;
- texto de direitos;
- ligações com outros registos.

A migração não afirma que a divisão legada entre estas camadas está correta. Por isso, todos os registos incluem `narrativeLayerReviewRequired: true`.

Textos com aspas ou testemunhos incorporados não foram convertidos automaticamente em `communityVoices`, pois faltam dados sobre autor, canal, data e consentimento específico.
