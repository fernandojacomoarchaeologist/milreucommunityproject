# Dados de exposições

O protótipo contém três fontes possíveis:

1. `data/itinerant-exhibitions.json`;
2. `data/itinerant-exhibitions.txt`;
3. dados inline em `exhibitions.js`.

As três apresentam eventos concretos, mas o próprio código classifica os dados inline como demonstração. Como o projeto está em evolução, todos os eventos devem ser tratados como `prototype-data` até confirmação explícita.

Nenhum evento foi migrado para um calendário oficial neste pacote.
