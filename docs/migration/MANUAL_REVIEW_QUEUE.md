# Fila de revisão manual

Todos os registos exigem revisão. A ordem inicial recomendada é:

1. `MM202617`, por estar bloqueado e ser derivado por IA;
2. `MM202618`, `MM202619` e `MM202631`, por intervenções digitais;
3. registos com pessoas parcialmente identificadas;
4. registos sem data ou com data de revelação;
5. restantes registos.
