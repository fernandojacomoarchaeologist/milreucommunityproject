# Decisões pendentes

Estas decisões não bloquearam a migração conservadora, mas bloqueiam etapas posteriores:

- localização definitiva dos masters de preservação;
- regra formal para distinguir consentimento comunitário, cedência e direitos de imagem;
- contacto público para pedidos de correção ou retirada;
- aprovação dos eventos da exposição;
- vocabulário controlado de tipos de memória;
- método de identificação pública de pessoas;
- fonte de verdade para narrativas novas;
- infraestrutura de persistência;
- estratégia para substituir ou manter a integração Drive;
- revisão final das quatro línguas;
- tratamento público das imagens assistidas por IA.

O Claude deve perguntar apenas quando uma destas decisões for necessária para a tarefa seguinte.
