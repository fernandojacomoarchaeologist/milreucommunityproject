# Escopo e objetivo

O Pacote 04 preserva a versão preliminar como fonte histórica e cria uma camada de dados estruturados para revisão futura.

## Incluído

- inventário dos ficheiros e funcionalidades;
- conversão dos 31 registos;
- checksums dos ativos;
- identificação de lacunas e ambiguidades;
- separação explícita entre migração mecânica e validação editorial;
- reversibilidade.

## Não incluído

- substituição de `museum-items.js` no site;
- publicação dos novos JSON;
- redefinição da interface;
- tradução de conteúdos;
- validação jurídica de direitos;
- identificação definitiva de pessoas;
- base de dados ou API;
- resolução automática de inconsistências.
