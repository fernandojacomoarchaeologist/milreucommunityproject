# Acessibilidade do Museu

- teclado em todas as rotas;
- Escape no modo imersivo;
- setas para navegação;
- controlos com nomes acessíveis;
- imagem com alternativa textual;
- contraste;
- foco;
- modo de lista;
- redução de movimento;
- informação disponível fora da fotografia.
