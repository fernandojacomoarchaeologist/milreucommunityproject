# Decisões pendentes não bloqueantes

Estas decisões não impedem a integração do Pacote 01. Tornam-se bloqueantes apenas quando o respetivo módulo for implementado.

| Tema | Decisão futura | Bloqueia qual trabalho |
|---|---|---|
| Repositório | Nome e URL definitivos | Publicação e documentação externa |
| Portal | Domínio e estrutura final de rotas | Deploy público |
| Direitos | Canal público para correção, contestação e remoção | Formulário e rodapés públicos |
| Jurídico | Validação institucional das políticas de direitos e consentimento | Publicação final de conteúdos sensíveis |
| Persistência | Serviço de base de dados e autenticação | Narrativas e mapa vivo |
| Infraestrutura | Plataforma de deploy do portal e do backend | Ambientes públicos |
| App | Conta de publicação, plataformas-alvo e ambiente cloud de desenvolvimento | Distribuição Flutter |
| Design | Acesso e análise detalhada do livro de Hauschild e Teichner | Pacote de design system |
| Traduções | Responsáveis pela revisão de inglês, espanhol e francês | Publicação multilingue revista |
| Dados | Regras finais de anonimização e release | Portal público de inquéritos |
| Biblioteca | Estratégia de armazenamento privado e acesso a PDFs | Visualizador de documentos protegidos |

## Regra

Claude deve questionar estes pontos apenas quando forem necessários para a tarefa atual. Não deve resolvê-los por escolha arbitrária.
