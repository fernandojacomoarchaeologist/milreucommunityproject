# O que o projeto é e não é

## O projeto é

- Um programa guarda-chuva de Arqueologia Pública e Comunitária.
- Uma investigação plurianual que combina participação, documentação, avaliação e devolução pública.
- Uma estrutura para iniciativas independentes, mas relacionadas.
- Um projeto aberto à revisão metodológica e à aprendizagem com resultados positivos ou negativos.
- Uma infraestrutura digital e física para consulta, participação e memória.
- Um espaço de ligação entre comunidade, sítio arqueológico, investigação, gestão e parceiros.
- Uma investigação viva, com versões, estados editoriais e decisões rastreáveis.

## O projeto não é

- Apenas o Museu de Memórias.
- Apenas uma página institucional ou promocional.
- Uma plataforma exclusivamente sobre escavações arqueológicas.
- Um substituto da entidade responsável pela gestão das Ruínas Romanas de Milreu.
- Um repositório indiscriminado de fotografias, testemunhos, PDFs ou dados pessoais.
- Um mecanismo de publicação automática de conteúdos enviados pelo público.
- Uma forma de transformar hipóteses, memórias ou dados preliminares em factos sem revisão.
- Uma autorização geral para reutilizar materiais de terceiros.
- Um sistema fechado: novas iniciativas podem ser propostas, desde que documentadas e avaliadas.

## Regra de fronteira

Quando uma tarefa puder pertencer a mais de uma iniciativa, deve ser classificada pelo seu objetivo principal e relacionada secundariamente às demais. Não duplicar dados apenas para manter cada iniciativa isolada.
