# Limites de dados pessoais

## Separação

- conteúdo público;
- dados operacionais;
- consentimentos;
- contactos;
- pedidos de direitos;
- auditoria.

## Princípio

Recolher apenas o necessário e publicar apenas o aprovado.

## Exportações

Snapshots públicos não podem conter:

- emails privados;
- telefones;
- endereços;
- IPs;
- tokens;
- notas internas;
- documentos de consentimento;
- coordenadas sensíveis.
