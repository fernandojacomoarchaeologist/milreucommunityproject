# SPEC-DS-005 — Laboratório de fundações

## Estado
Approved.

## Objetivo
Permitir inspeção interna de cor, tipografia, espaçamento, grelha, formas, movimento, fotografia e dados antes do catálogo 05C.

## Dentro do âmbito
- leitura dos tokens v0.2;
- navegação por secções;
- swatches e contraste;
- amostras tipográficas;
- grelha e espaçamento;
- simulação de abertura vermelha, leitura clara e destaque preto;
- estado interno claramente visível.

## Fora do âmbito
- documentação completa de componentes;
- publicação pública;
- reprodução de páginas do livro;
- editor de tokens;
- logótipo;
- exemplos do Museu com dados reais.

## Critérios de aceitação
- funciona sem framework;
- mantém leitura se fontes web falharem;
- responde a desktop e mobile;
- não contém imagens do livro;
- utiliza os tokens consolidados;
- respeita redução de movimento.
