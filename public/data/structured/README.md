# Dados estruturados

Este diretório contém uma exportação JSON-LD do Museu.

## Referências técnicas

- JSON-LD 1.1 — W3C Recommendation: https://www.w3.org/TR/json-ld11/
- Schema.org `Dataset`: https://schema.org/Dataset
- Schema.org `Photograph`: https://schema.org/Photograph
- Schema.org `ImageObject`: https://schema.org/ImageObject

## Limite

O domínio público ainda não foi definido. Os identificadores usam URNs e as URLs de conteúdo permanecem relativas até à publicação.
