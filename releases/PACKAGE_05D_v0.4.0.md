# Release — Pacote 05D v0.4.0

## Adicionado

- 21 componentes especializados;
- 13 padrões de composição e fluxo;
- bibliotecas CSS e Vanilla JS;
- ativos SVG exclusivamente demonstrativos;
- páginas de componentes com anatomia, variações, estados, acessibilidade, tokens e plataformas;
- páginas de padrões com componentes utilizados, fluxo, riscos e limites;
- documentação, specs, rules, skills e testes;
- handoff para o próximo pacote.

## Alterado

- catálogo visual atualizado de 0.3.0 para 0.4.0;
- páginas `components/overview` e `patterns/overview` passam a inventários funcionais;
- release do guia passa a representar a conclusão do ciclo 05.

## Não incluído

- Portal ou Museu públicos;
- backend e persistência;
- utilização automática dos 31 registos migrados;
- aplicação Flutter;
- templates finais de impressão;
- logótipo final;
- promoção de elementos para `approved`.

## Migração

Atualizar `apps/design-guide/`, adicionar os três pacotes v0.4 e integrar documentação e regras. O caminho canónico permanece `apps/design-guide/`.

## Estado

`internal-preview`.
