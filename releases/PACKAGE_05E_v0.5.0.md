# Release — Pacote 05E v0.5.0

## Adicionado

- original preservado do logótipo;
- variantes raster transparentes para fundos claros e escuros;
- versões monocromáticas e símbolo isolado;
- favicon, ícones de aplicação e previews sociais;
- política de cores chapadas;
- arquitetura de marca Projeto–iniciativas–Milreu Proteus;
- 16 ícones funcionais e 16 ícones de domínio;
- tokens, manifests, documentação, specs, rules, skills e testes;
- pré-visualização independente e extensão do Design Guide;
- handoff para o Pacote 06.

## Alterado

- o ciclo 05 passa de 0.4.0 para 0.5.0;
- o logótipo deixa de ser uma pendência abstrata e passa a ter ativos `draft` revistos por regras explícitas;
- Milreu Proteus é formalizado como base de conhecimento subordinada.

## Não incluído

- SVG oficial ou vetorização aprovada;
- assinatura horizontal;
- promoção dos ativos para `approved`;
- alteração do nome do site;
- logótipo próprio para Proteus ou Museu;
- Portal, Museu, backend ou aplicação Flutter.

## Estado

`internal-preview`.
