# site/ — LACUNA ASSUMIDA

Esta pasta está vazia por decisão, não por esquecimento.

O site comunitário (`index.html`, `style.css`, `main.js`, `exhibitions.js`,
`drive-integration.js`) é um **sistema vivo**: tem integração com o Google
Drive e um formato `.txt` que a comunidade já preenche.

Esse código **não deve ser reconstruído de memória por nenhum agente**.
Recriá-lo produziria algo que não é o que está em produção e partiria o
fluxo real das pessoas que contribuem.

Entra no repositório por um pack de importação, quando os originais
estiverem disponíveis.
