# Mapa de tokens

| Token preliminar | Alias inicial | Justificação |
|---|---|---|
| `--color-parchment` | `--milreu-paper` | superfície principal |
| `--color-parchment-dark` | `--milreu-paper-deep` | superfície secundária |
| `--color-parchment-mid` | mistura Paper/Paper Deep | compatibilidade |
| `--color-terracotta` | `--milreu-sepia` | evita utilização extensa de tijolo |
| `--color-terracotta-dark` | `--milreu-ink-soft` | estados escuros sem institucionalizar a superfície |
| `--color-terracotta-light` | `--milreu-stone` | acento suave |
| `--color-olive` | `--milreu-patina` | acento comunitário |
| `--color-olive-light` | mistura Patina/Paper | compatibilidade |
| `--color-olive-dark` | mistura Patina/Ink | compatibilidade |
| `--color-sand` | `--milreu-stone` | filetes e limites |
| `--color-sand-light` | `--milreu-paper-deep` | limites claros |
| `--color-charcoal` | `--milreu-ink` | texto principal |
| `--color-charcoal-mid` | `--milreu-ink-soft` | texto intermédio |
| `--color-charcoal-light` | `--milreu-ink-soft` | texto secundário |
| `--color-white` | `--milreu-paper` | branco quente do sistema |
| `--font-display` | `--ds-font-display` | Fraunces |
| `--font-body` | `--ds-font-body` | Spectral inicial |
| `--font-accent` | `--ds-font-display` | eliminação progressiva do terceiro papel tipográfico |

## Nota

Este mapa preserva funcionamento, não garante resultado final. Classes que utilizam cores antigas por função inadequada devem migrar para tokens semânticos.
